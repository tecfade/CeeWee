## Description

Entwicklung eines interaktiven Dashboards zur Visualisierung und Auswertung von Echtzeit-Geschäftsdaten.

Umsetzung einer komponentenbasierten Benutzeroberfläche mit React und TypeScript, dokumentiert in Storybook.

Anbindung einer GraphQL-API inkl. automatischer Typgenerierung für typsichere Datenzugriffe.

Sicherstellung hoher Codequalität durch Unit-Tests (Jest), End-to-End-Tests (Cypress) und statische Analysen in der GitLab-CI-Pipeline.

Enge Abstimmung mit UX-Design und Product Owner zur iterativen Weiterentwicklung der Features auf Basis von Nutzerfeedback.
